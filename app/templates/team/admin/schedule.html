How to schedule people

Fast path: pick the week, tap Edit schedule, add the employee to the roster if needed, click their day cell, choose the shift type, set times/notes, then save the storefront schedule.

  1. Pick the week: Prev, Next, or Jump to this week. Weeks start Monday.
  2. Turn on editing: Edit schedule makes cells clickable and shows roster controls.
  3. Add roster people: use the roster picker. If someone is missing, mark them On schedule on Employees.
  4. Click a cell: choose Shift, OFF, SHOW, REQUEST, Stream, or IF NEEDED in the edit sheet.
  5. Set time and repeat: enter start/end, add a note, or use repeat/generate from last week when it saves time.
  6. Save: unsaved changes get a dot. Storefront and Stream grids save separately.
