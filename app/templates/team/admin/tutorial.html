How to schedule people

The schedule page opens read-only on purpose. Editing is a deliberate mode so managers do not accidentally change a week while checking staffing.

  1. Pick the week first. Use Prev, Next, or Jump to this week. The week always runs Monday through Sunday.
  2. Tap Edit schedule. Edit mode turns cells into buttons and shows roster, save, discard, and generate controls.
  3. Add the right people to the roster. Use the roster picker above Storefront or Stream. If someone is missing, open Employees and turn on On schedule, then set their Type.
  4. Click a person/day cell. The edit sheet opens. Choose Shift, OFF, SHOW, REQUEST, Stream, or IF NEEDED.
  5. Set times and a short note. Use 15-minute time steps. Notes are good for opener, front counter, studio, runner, or who they are with.
  6. Use repeat or previous week when it saves time. Repeat clones one shift forward. Copy previous week adds last week's roster. Generate from last week clones shifts while preserving existing cells.
  7. Save the schedule. Unsaved cells get a dot. Storefront saves with Save storefront schedule. Stream schedule changes sync with Stream Manager.

Schedule statuses

Shift A normal working shift. This is what most employees should have.
OFF A deliberate off day so managers know the blank is not an accident.
SHOW A special event or show day. Useful when the location or normal routine changes.
REQUEST Time-off or request marker. Check Time off before overriding this.
Stream A live stream shift. These sync with the Stream Manager and stream account setup.
IF NEEDED A soft backup shift. Use it when someone might be called in but is not fully scheduled yet.

Best daily manager flow

1. Overview Start with Needs attention so supply, time off, password reset, and timecard problems do not pile up.
2. Time off Approve or deny requests before editing the schedule. This keeps REQUEST days meaningful.
3. Schedule Build the week, save it, then have employees check their Schedule page.
4. Shift Tracker During the day, watch clock status, breaks, and current hours from Clockify.
5. Announcements Post a team update when schedule changes, policies, or shop-floor changes affect everyone.
